# 💻 Introdução à Ciência da Computação

Este repositório foi criado para armazenar atividades, exercícios, anotações e projetos desenvolvidos durante a disciplina **Introdução à Ciência da Computação**.

O objetivo é organizar os conteúdos estudados ao longo do semestre, registrar minha evolução na área da computação e reunir materiais que possam servir como consulta futura.

## 📚 Sobre a disciplina

A disciplina de Introdução à Ciência da Computação apresenta conceitos fundamentais da área, permitindo compreender melhor como os computadores funcionam, como os algoritmos são construídos e como a programação pode ser utilizada para resolver problemas.

Entre os principais temas estudados, estão:

- Conceitos básicos de computação;
- História e evolução dos computadores;
- Pensamento computacional;
- Algoritmos;
- Lógica de programação;
- Estruturas básicas de programação;
- Representação de dados;
- Resolução de problemas computacionais.

## 🎯 Objetivos do repositório

Este repositório tem como finalidade:

- Registrar os conteúdos estudados na disciplina;
- Armazenar exercícios e atividades práticas;
- Organizar códigos desenvolvidos durante as aulas;
- Acompanhar minha evolução no aprendizado de computação;
- Servir como material de revisão e consulta.

## 🛠️ Tecnologias e ferramentas

Durante a disciplina, poderão ser utilizadas ferramentas como:

- Editor de código;
- Linguagem de programação introdutória;
- Git e GitHub;
- Ambientes de desenvolvimento online ou local;
- Materiais de apoio disponibilizados pelo professor.

## 🧠 Aprendizados

Ao longo da disciplina, este repositório será atualizado com novos conteúdos e práticas. A ideia é demonstrar o progresso no entendimento dos conceitos iniciais da computação, principalmente no desenvolvimento do raciocínio lógico e na criação de soluções por meio de algoritmos.

## 🚧 Status do repositório

Em desenvolvimento.

Novos arquivos, exercícios e projetos serão adicionados conforme o andamento da disciplina.
